# @rcarmo/piclaw-addon-cheapskate

Free-tier provider auto-rotation — select cheapskate/auto as your model and it transparently routes to Google Gemini, Cerebras, Groq, or SambaNova free tiers

## Install

Open **Settings → Add-Ons** and install **cheapskate** from the catalog.

- Agent skills for guided tool usage
